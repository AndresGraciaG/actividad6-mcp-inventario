# Actividad 6 - MCP Inventario

Servidor MCP desarrollado en Python utilizando FastMCP y SQLite para la gestión de inventario mediante operaciones CRUD y consultas analíticas.

- Instalación de librerías: pip install -r requirements.txt

- Ejecución: python server.py

# Explicación: 
El objetivo de esta actividad es diseñar e implementar un servidor MCP en Python que permita realizar operaciones CRUD y consultas analíticas sobre una base de datos de inventario, con el propósito de comprender cómo un sistema cognitivo puede interactuar con herramientas externas.
